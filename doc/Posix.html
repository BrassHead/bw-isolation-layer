Cross Reference to POSIX

The posix standard (IEEE Std 1003.1-1988) for operating system interface has "C" routines in the areas of Processes (including Signals), Process environment, Files and Directories, I/O Primitives, Device Specific functions, C Language Services, and System Databases. A separate standard covers POSIX Threads.

Within the BW framework, POSIX routines are handled by classes specific to different operating system resources as follows:

Process primitives: process.h
Process environment: process.h
Files and directories: directory.h, filename.h
I/O primitives: file.h
Device specific functions: n/a
C language services: process.h
System databases: n/a
POSIX threads: T.B.D.
