Win32 platform notes

Compiler options for VC++:
  • Preprocessor definitions: WIN32, _WINDOWS
  • Enable exception handling
  • Enable RTTI
  • Disable language extensions (actually NO, since <iostream> needs the language extensions (how strange!)
  • Additional include directories: d:\src\BW
  • Add "d:\src\BW\[Debug|Release]\BW.lib" to start of library modules
  • Warning level 3 (Recommended)
  • Enable incremental compilation (Recommended)
