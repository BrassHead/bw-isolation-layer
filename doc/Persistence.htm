Databases

To be written.

The intent is to model the classes on SQL/CLI (aka ODBC). Candidate objects are:

  • Database -- represents a connection to a database
  • Statement -- represents an SQL statement with methods for binding parameters
  • RowSet -- result of executing a statement
  • Row -- a base class for individual results. A table in the subclass defines the binding between SQL fields and object members.
