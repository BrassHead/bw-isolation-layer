LineTool

Base class for line Drawing objects.

LineTool::setDashes()

void LineTool::setDashes( int on, int off )

Sets the tool for drawing dashed lines. The two parameters give the dash on and off lengths in pixels. If 'off' is 0, then a solid line is drawn. This is the default for a new LineTool.
