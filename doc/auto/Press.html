Drawing object for text.

Press::Press()

Press::Press( Canvas& cvs, FontIndex findx=FaceFont )

Constructors.

The parameters are the Canvas to draw on and optionally the font of the text face. If no font is specified, the default foreground Text font will be used. The text colour is set to the default foreground text colour and the text background colour is set to the default background.

Changes the font used for printing.

Press::drawText()

void Press::drawText( int x, int y, String str )

Draws the given string. Only the pixels within the font are affected (no background box is drawn).
