LogStream

Output stream for event logging.

This is a simple logging stream. It doesn't interface to syslog() and expects good co-operation with the caller.

Example:
LogStream lout; lout(ERR) << "An error message." << endl;
Rules:
Ident should be program name Don't send '\n's Always end message with endl
Output:
Outputs lines starting with the date and time followed by the ident: and the message.
