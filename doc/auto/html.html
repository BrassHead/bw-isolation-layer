Ostream manipulators for writing HTML files.

These manipulators send HTML formatting codes to an ostream.

The prolog and epilog manipulators write everything necessary to start and end the BODY of an HTML document.

html::rule inserts a horizontal rule.

html::literal( foo ) outputs foo expanding <'s, "'s and &'s so that they appear literally in the output and are not interpreted as HTML commands. Newlines are converted to "
" directives if isMultiLine is true.

html::smartFormat( foo ) inserts paragraph markers into foo wherever blank lines are found.

Here are a few examples to clarify the other ones that are not self evident:

Definitions:

os << html::beginDefinitionList;
os << html::definition( "Item1" );
os << "The definition of Item1";
os << html::definition( "Item2" );
os << "The definition of Item2";
...
os << html:endDefinitionList;
Links:

os << html::beginLink( "http://www.somewhere.com/" );
os << "Click here";
os << html::endLink;


os << html::defineLink( "top" );
os << "Some text.";
...
os << "Blah blah blah...";
os << html::beginLink2Link( "top" );
os << "Return to top of page.";
os << endLink;
