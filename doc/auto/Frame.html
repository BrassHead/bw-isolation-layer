Frame::onClose()

void Frame::onClose()

Overidable interface to Frame Figures.

This routine is called when the user has selected the "close" or "quit" or whatever button, menu, and/or key from their host OS.

The correct response is to cleanly exit. Additional dialog boxes can be raised to query the user about saving data, etc.

The default action is 'doAction("exit(0)")'
