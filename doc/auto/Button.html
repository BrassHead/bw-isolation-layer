Button

This is a basic button Figure. It is written as a true BW figure without using a native control.

Button is a subclass of Figure, Focus, and Mouse and defines event handling functions declared in these base classes.

Button::draw()

void Button::draw( Canvas& cvs )

Draws button. This buttons style has quite generic 3D look. Focus is indicated by a dashed line just inside the button face. The standard style colours (Face, ButtonFace, etc) are used. The default button is indicated with a black border.
