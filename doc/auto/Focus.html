Focus

This is a mix-in class for Figures that indicates that the Figure accepts keyboard input and/or it can be the input focus for the application.

Focus member functions

Focus() Constructor.
releaseFocus() Causes the Figure to cease to be the input focus.
tabFocus() Moves the input focus to the next Figure in the current tab group.
takeFocus() Causes the Figure to become the active input focus.
~Focus() Destructor.
